# Performance Evaluations of Mixed-Precision RK Methods 

This repository contains the Julia code to reproduce the numerical experiments from the paper "Performance Evaluation of Stabilized Corrections for Mixed Precision Runge–Kutta Methods" (https://arxiv.org/pdf/2607.10967).

## Folders
* Burgers: Contains the codes for Burgers' equation
* PM: Contains the codes for the porous medium equation
