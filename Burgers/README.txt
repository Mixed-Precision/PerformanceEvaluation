# Mixed-Precision RK Methods for Burgers' Equation

This folder contains the Julia code to reproduce the numerical experiments for Burgers' equation from the paper "Performance Evaluation of Stabilized Corrections for Mixed Precision Runge–Kutta Methods" (https://arxiv.org/pdf/2607.10967). 

## Author
César Herrera 
Email: herre125@purdue.edu

## Julia Files
* time.jl: Driver script to generate the time vs. error plots.
* speedup_double.jl: Driver script to generate speedup tables for double precision.
* speedup_quad.jl: Driver script to generate speedup tables for quad precision.
* BurgersMP.jl: The mathematical core containing the Fourier spectral spatial discretization and RK methods in both full and mixed precision.
* BurgersExperiments.jl: The experiment logic, reference solution generation, error calculations, timing, and plotting routines.

## Folders
* Times: Contains data resulting from different experiments. It also stores .xlsx files generated by the speedup drivers.
* Solutions: Reference solutions for various Nx.
* Results: Time vs. error plots are saved in this folder.
